About Me

I am Keerthi Krishna Kadali, currently pursuing a degree in Electronics and Communication Engineering with a strong interest in software development. My technical skills include HTML, CSS, JavaScript, Basic C, and Basic Python, and I enjoy building web-based solutions that combine both creativity and problem-solving.

I have worked on a project titled Smart Agriculture Information System, developed using HTML, CSS, JavaScript, and Python (Flask). This web platform provides farmers with crop details, irrigation methods, fertilizers, pesticides, and damage control suggestions. It also integrates multi-language voice/text search and an AI voice assistant, making agricultural knowledge more accessible.

My career goal is to become a software developer, contributing to innovative projects that improve everyday life through technology.
